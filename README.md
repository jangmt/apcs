```
BMI標準
男生：10以下(太瘦)、11-21(標準)、22以上(過高)
女生：20以下(太瘦)、21-24(標準)、25以上(過高)

1.開始
2.輸入性別(1男2女) gender
3.輸入體重 weight
4.輸入身高 height
5.計算BMI值
6.假如 性別==1:
        假如 BMI<=21:
                輸出BMI值, 顯示"標準"
        否則:
                輸出BMI值, 顯示"過高"
   否則:
        假如 BMI<=24:
                輸出BMI值, 顯示"標準"
        否則:
                輸出BMI值, 顯示"過高"
```
